# Adapters

Integrações opcionais vivem neste diretório. O núcleo não depende destes adapters para funcionar.

Adapters disponíveis nesta instância:

- `llm`: integrações opcionais com modelos
- `n8n`: ponto de integração futuro com automação
- `storage`: persistência externa opcional
- `ruptur-forge`: camada plan-only de catálogo e perfis de bootstrap
